## How to re-run the gate

```bash
# 1. Replay all fixtures + regenerate matrix + marker
npx tsx scripts/completion-gate.ts

# 2. Cross-validate the matrix JSON (8 conditions)
bash scripts/check-evidence.sh artifacts/phase-6/completion-matrix.json

# 3. Verify every assertion passes
node -e "const m=require('./artifacts/phase-6/completion-matrix.json'); const fail=m.assertions.filter(a=>!a.pass); process.exit(fail.length===0?0:1)"

# 4. Vitest contract tests (structure + author signoff + marker)
npx vitest run test/webview/completion-gate.test.ts
```

All four must exit 0. If any fails, `scripts/completion-gate.ts`
refuses to emit the marker and the iteration is considered incomplete.
